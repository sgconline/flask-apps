This application has been written to assist me in my learning journey.

The application was written using Python and the Flask Web Framework. The application is hosted by gunicorn with nginx acting as a reverse proxy all within the same image.

The application is a simply address book in which contacts can be added or removed from the backend json db.

Next versions of the application will include splitting out nginx into it's own container, rewritting the application to use a MongoDB backend store

Whilst upskilling on the following technologies:
  • Git and GitHub all functionality of the address book application is versioned correctly in git

  • Python programming, following the course on Codecademy

  • Docker including image building and pushing to DockerHub

  • Kubernetes Administration and development
